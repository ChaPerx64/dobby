# Dobby feature requirements

# Description

Dobby is a household management service

## Finances management

It should help with managing finances and have the following functionality:

It should have financial periods (usually, abound a month)

In any given period, it allocates finances to preset users and purposes.
For now, two users: TheMan and TheWoman
The household gets 60'000 RSD per period for groceries and housekeeping
Each user (TheMan and TheWoman) gets 30'000 RSD each
